# Languages / Translations

This folder contains translation files for the Greek VAT & Invoices for WooCommerce plugin.

## Translation Files

- **greek-vat-invoices-for-woocommerce.pot** - Template file for creating translations

## How to Translate

### Method 1: Using Loco Translate Plugin (Recommended)

1. Install and activate [Loco Translate](https://wordpress.org/plugins/loco-translate/) plugin
2. Go to **Loco Translate → Plugins**
3. Find "Greek VAT & Invoices for WooCommerce"
4. Click "New language"
5. Select your language and location
6. Start translating strings
7. Save and sync

### Method 2: Using Poedit

1. Download and install [Poedit](https://poedit.net/)
2. Open the `.pot` file in Poedit
3. Create a new translation (File → New from POT file)
4. Select your language
5. Translate all strings
6. Save as `.po` and `.mo` files
7. Upload both files to this `languages` folder

### Method 3: WordPress.org Translation Platform

Once the plugin is approved on WordPress.org, you can contribute translations at:
https://translate.wordpress.org/projects/wp-plugins/greek-vat-invoices-for-woocommerce/

## File Naming Convention

Translation files should be named:
- `greek-vat-invoices-for-woocommerce-{locale}.po`
- `greek-vat-invoices-for-woocommerce-{locale}.mo`

Examples:
- `greek-vat-invoices-for-woocommerce-el_GR.po` (Greek - Greece)
- `greek-vat-invoices-for-woocommerce-en_US.po` (English - USA)
- `greek-vat-invoices-for-woocommerce-de_DE.po` (German - Germany)

## Current Translations

- **Greek (Ελληνικά)** - Built-in, strings are already in Greek
- Other languages - Use the methods above to create translations

## Text Domain

The plugin uses the text domain: `greek-vat-invoices-for-woocommerce`

All translatable strings in the code use this text domain for proper internationalization.
